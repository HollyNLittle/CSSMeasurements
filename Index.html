CSS MEASUREMENTS

rem, em, vh and vw

Two rem

  • Pixels are fixed.
  • Rem and em are relative.
  • Rem/em can be controlled by the user in the browser settings, appearance, font-size.
  • Rem/em are multipliers and can be used on elements other than text.

Three rem

  • Pixels are fixed. 2rem
  • Rem and em are relative. 4rem
  • Rem/em can be controlled by the user in the browser settings, appearance, font-size. 2em
  • Rem/em are multipliers and can be used on elements other than text. 4em

Rem is relative to the root element, (the HTML element) which is by default, 16px. 1rem = 16px. Rem is a multiplier, if set font size of body text to 10px, then 2 rem will be 20 pixels, can use decimals, 1.5rem will be 15 pixels and 6rem would be 60 pixels. Rem can be used for other elements as a multiplier, border:1rem or 16 pixels or use decimals border:0.5rem would be 8 pixels.


Em is relative to the parent element. Font-size 10px as the parent. If parent element has a fixed, defined size, like 15px, then that will override rem/em.

height:20vh/width: 20vw

  • VH/VW are viewpoint height/viewpoint width
  • 20vh would be 20/100 viewpoint heights
  • 100VH/VW = 100% of screen
  • 50VH = half of the page horizontally
  • 50VW = half of the page vertically
